# Base Python Project
### all set to start from scratch 8-)
---

#### ___its dangerous to go alone, take this:___

1. Setup:
    ```
    python -m venv .env
    ```
2. Activate:
    ```
    .env\scripts\activate
    ```
3. After Install:
    ```
    pip freeze > requirements.txt
    ```
4. On Another Machine:
    ```
    pip install -r requirements.txt
    ```
    
### Check Yourself:
    pip list
